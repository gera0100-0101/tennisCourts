# Padel Court Installation Website

Modern business card website for a company specializing in padel court installation.  
Frontend built with **React.js + Vite**, backend powered by **ASP.NET Core**.

## 🚀 Tech Stack

### Frontend
- React.js
- Vite
- JavaScript / JSX

### Backend
- ASP.NET Core Web API
- Swagger

---

## 📌 About the Project

Business website for a company providing:

- Turnkey padel court installation
- Sports facility design
- Surface installation
- Lighting systems
- Fencing systems
- Maintenance and repairs

---

## ⚙️ Main Features

### Frontend

- Home page
- About company
- Services page
- Project gallery
- Contact form
- Contact information

### Backend

- REST API
- Contact form request handling
- Email notifications

## 🛠 Installation & Run

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
dotnet restore
dotnet run
```

## 🐳 Run with Docker

The entire project can be started using **Docker** and **Docker Compose**.

### Start Containers

```bash
docker-compose up --build
````

### Run in Background

```bash
docker-compose up -d
```

### Stop Containers

```bash
docker-compose down
```

### Services

* **client** — React.js + Vite frontend
* **server** — ASP.NET Core Web API

### Notes

Make sure **Docker** and **Docker Compose** are installed before running the project.
